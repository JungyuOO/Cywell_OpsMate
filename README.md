# Cywell_OpsMate

Cywell OpsMate is planned as a Go-based OpenShift Operator for OpenShift Web Console integration, Lightspeed API usage, AIOps capabilities, and optional RAG expansion.

Development is gated by versioned specs under `spec/<version>/`. The current `v0.0.0` scope is limited to project rules, Lightspeed Operator structure review, and an initial Go Operator scaffold.

## Current Commands

```bash
make fmt
make test
make build
```

These commands require Go to be installed locally.
